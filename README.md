# python_github_assignment
# TITLE: Daily study tracker

# Description
This is a simple program that lets you type in how many hours you studied today. It uses that number to estimate how much you’ll study in a full week if you keep the same pace. It is a quick way to see if you’re on track or falling behind. The program is easy to use and only needs one number from you.

# how to run
python app.py 
or
python3 app.py

Welcome to the Study Tracker!
How many hours did you study today? 3
If you study like this every day, you'll study about 21.0 hours this week.
